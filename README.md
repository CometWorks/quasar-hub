# QuasarHub
 
This repository contains plugins for [Quasar](https://github.com/CometWorks/quasar), 
a Space Engineers Dedicated Server control plane.

## How to add a plugin?

If you wish to make a plugin for Quasar, there are a few steps:
1. Use this template: [**Quasar Plugin Template**](https://github.com/CometWorks/quasar-plugin-template) (click the green button in the top right corner).
2. Publish your code on GitHub. It must be in a public repository once you want to publish it. 
   It is perfectly okay to initially work in a private repository, put you must set the repository public to get a review.
3. Add your XML plugin descriptor file to the `Plugins` folder in this repository. Example file: [SamplePlugin.xml](SamplePlugin.xml)
4. Submit a pull request. The approval team will approve or deny it.
